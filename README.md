# Nano Matching Engine (C++20)

A minimal, single-file order matching engine built in modern C++20.  
Implements a lock-free producer/consumer queue and a price time priority order book.



## Build
g++ -O3 -std=c++20 nanobook.cpp -o nanobook -pthread
